# Contributing

Thank you for your interest in improving this fork of [openmm/pdbfixer](https://github.com/openmm/pdbfixer). To keep the
project compliant with the upstream MIT License, please follow these guidelines:

- By submitting a contribution, you agree that your changes are licensed under the repository MIT License.
- Do not remove or alter existing copyright or license headers; preserve upstream attribution when modifying files.
- Add a brief "Modified by" note if you significantly change upstream-derived files and ensure new files include an MIT
  license header or are clearly covered by the repository LICENSE file.
- Keep changes focused and include tests or documentation updates when applicable.
- Open an issue or discussion if you are unsure about licensing or attribution for a change.
